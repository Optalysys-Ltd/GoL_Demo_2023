Open\+FHE -\/ Open-\/\+Source Fully Homomorphic Encryption Library

Fully Homomorphic Encryption (FHE) is a powerful cryptographic primitive that enables performing computations over encrypted data without having access to the secret key. Open\+FHE is an open-\/source FHE library that includes efficient implementations of all common FHE schemes\+:
\begin{DoxyItemize}
\item Brakerski/\+Fan-\/\+Vercauteren (BFV) scheme for integer arithmetic
\item Brakerski-\/\+Gentry-\/\+Vaikuntanathan (BGV) scheme for integer arithmetic
\item Cheon-\/\+Kim-\/\+Kim-\/\+Song (CKKS) scheme for real-\/number arithmetic (includes approximate bootstrapping)
\item Ducas-\/\+Micciancio (DM) and Chillotti-\/\+Gama-\/\+Georgieva-\/\+Izabachene (CGGI) schemes for Boolean circuit evaluation
\end{DoxyItemize}

Open\+FHE also includes the following multiparty extensions of FHE\+:
\begin{DoxyItemize}
\item Threshold FHE for BGV, BFV, and CKKS schemes
\item Proxy Re-\/\+Encryption for BGV, BFV, and CKKS schemes
